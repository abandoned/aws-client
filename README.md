# AWS Client

[![travis] [status]] [travis]

Built on [Faraday] [faraday] and [Nokogiri] [nokogiri], AWS Client is an abstract Ruby client to [Amazon Web Services] [aws].

See [Vacuum] [vacuum] for an implementation.
